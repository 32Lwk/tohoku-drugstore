# 愛知県 ドラッグストア調査

愛知県内のドラッグストア調査プロジェクト（`愛知県内のドラックストア`）の成果物を取り込み。

## データ

- `data/愛知県ドラッグストア_最終版.csv` — クリーニング済み店舗一覧
- `data/愛知県ドラッグストア_座標付き.csv` — 座標付き店舗一覧
- `data/市区町村別ドラッグストア分析.csv` — 密度分析
- `data/市区町村別人口.csv` — 人口・面積
- `data/市区町村別高齢化率.csv` — 高齢化率
- `data/municipalities.geojson` — 市区町村境界
